# Employee Attendance Automation System

## Project Overview

This project is an Employee Attendance Automation System built in Google Sheets.  
The main goal was to reduce manual work by automating attendance entry, employee lookup, work hour calculation, late minute calculation, overtime tracking, and dashboard reporting.

## Tools Used

- Google Sheets
- Google Sheets Macros
- Google Sheets Formulas
- XLOOKUP
- IF Function
- COUNTIF
- SUM
- AVERAGE
- Dashboard Design

## Key Features

- Automated attendance entry form
- Dynamic employee lookup using Employee ID
- Automatic employee name and department retrieval
- Automatic work hour calculation
- Automatic late minute calculation
- Automatic overtime hour calculation
- Attendance status detection
- One-click data submission using Google Sheets Macros
- Structured attendance database
- Interactive attendance dashboard

## Project Structure

The project contains four main sections:

1. **Employee List**  
   Stores employee master data including Employee ID, Employee Name, Department, Shift Start, and Shift End.

2. **Attendance Entry Form**  
   Used to enter daily attendance information.

3. **Attendance Database**  
   Stores all submitted attendance records in a structured format.

4. **Dashboard**  
   Provides summary insights using KPI cards and charts.

## Dashboard KPIs

- Total Attendance Records
- Present Count
- Late Count
- Total Overtime Hours
- Average Work Hours

## Dashboard Charts

- Attendance Status Distribution
- Department-wise Attendance

## Business Value

This system helps reduce repetitive manual work, improves reporting accuracy, and provides quick attendance insights for management decision-making.
